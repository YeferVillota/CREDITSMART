## Descripción
CREDITSMART es una aplicación para solicitar creditos, en ella puedes encontrar los difierentes tipos de creditos disponibles y solicitar el que necesites, puedes realizar simulacros para determinar el credito de tu preferencia antes de solicitar uno real, y una vez te hayas decidido, puedes realizar tu credito despues de llenar un formulario.
